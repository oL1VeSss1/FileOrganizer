# 📂 File Organizer (Python)

## 📌 Description
**File Organizer** — это консольная программа на Python для работы с файлами и папками.  
Она помогает:
- создавать папки
- просматривать содержимое директорий
- перемещать файлы
- сортировать файлы разными способами

Проект отлично подходит для практики работы с файловой системой в Python.

---

## ⚙️ Features

### 📁 Работа с папками
- Проверка существования пути
- Создание новых папок
- Просмотр содержимого директории

### 🔄 Работа с файлами
- Перемещение файлов из одной папки в другую

### 📊 Сортировка файлов
- По размеру
- По расширению (категориям)
- По алфавиту (A–Z / Z–A)

---

## 🧠 File Categories

Программа автоматически группирует файлы по типам:

- Images (.jpg, .png, .jpeg, .ico, .gif, .webp, .bmp)
- Documents (.pdf, .docx, .txt, .csv)
- Music (.mp3, .wav)
- Videos (.mp4, .avi)
- Archives (.zip, .rar)
- Executable files (.exe, .dll, .com, .bat)
- Programming languages (.py, .js, .ts, .java, .cpp, .cs, .rs, .go)
- Other (все остальное)
