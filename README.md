# 📂 File Organizer (Python)

## 📌 Description
**File Organizer** is a console-based Python program for working with files and folders.  
It allows you to:
- create folders
- browse directory contents
- move files
- sort files in different ways

This project is great for practicing file system operations in Python.

---

## ⚙️ Features

### 📁 Folder Management
- Check if a path exists
- Create new folders
- View directory contents

### 🔄 File Operations
- Move files from one folder to another

### 📊 File Sorting
- By size
- By file extension (categories)
- Alphabetically (A–Z / Z–A)

---

## 🧠 File Categories

The program automatically groups files into categories:

- Images (.jpg, .png, .jpeg, .ico, .gif, .webp, .bmp)
- Documents (.pdf, .docx, .txt, .csv)
- Music (.mp3, .wav)
- Videos (.mp4, .avi)
- Archives (.zip, .rar)
- Executable files (.exe, .dll, .com, .bat)
- Programming languages (.py, .js, .ts, .java, .cpp, .cs, .rs, .go)
- Other (everything else)

---

## 🚀 How to Run

1. Make sure you have Python installed (3.x)
2. Download the project file
3. Run in terminal:

```bash
python "File organizer.py"
```
